# Pipeline de Previsão do Tempo

Projeto de estudos para aprender conceitos de Engenharia de Dados utilizando Python para coleta e processamento de dados e Laravel para disponibilização das informações por meio de uma API e interface web.

## Objetivos

- Aprender consumo de APIs com Python.
- Construir um pipeline de dados simples.
- Armazenar dados meteorológicos em banco de dados.
- Expor os dados através de uma API Laravel.
- Praticar integração entre Python e Laravel.

## Tecnologias

### Python

- Python 3
- Requests
- Pandas
- SQLAlchemy
- python-dotenv

### Backend

- Laravel
- PHP
- MySQL ou PostgreSQL

## Fluxo

1. O Python consulta uma API de previsão do tempo.
2. Os dados são tratados e normalizados.
3. As informações são gravadas no banco de dados.
4. O Laravel lê esses dados.
5. A API e a interface exibem as previsões ao usuário.

## Objetivos de Aprendizagem

Durante o projeto serão abordados conceitos como:

- Consumo de APIs REST
- Estruturação de projetos Python
- ETL (Extract, Transform, Load)
- Modelagem de banco de dados
- Agendamento de tarefas
- Integração entre aplicações
- Boas práticas de organização de código

## Evolução do Projeto

O desenvolvimento será dividido em ciclos.

- Ciclo 1 — Estrutura do projeto
- Ciclo 2 — Extração dos dados
- Ciclo 3 — Transformação
- Ciclo 4 — Persistência no banco
- Ciclo 5 — API Laravel
- Ciclo 6 — Dashboard
- Ciclo 7 — Automatização
- Ciclo 8 — Melhorias e deploy
